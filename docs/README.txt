📚 AirSense – Dokümantasyon Bölümü

Bu klasör, proje boyunca oluşturulan tüm dökümantasyon, rapor, kılavuz ve tanıtım materyallerini içerir.

🧩 İçerik:
- Teknik dökümanlar (sistem mimarisi, sensör yapısı, veri akışı)
- Kullanım kılavuzu (cihaz kurulumu, uygulama bağlantısı)
- Test raporları ve ölçüm sonuçları
- Proje sunumu veya broşür (PDF, PowerPoint)
- API dokümantasyonu (JSON veri yapısı, endpoint açıklamaları)

📘 Format Önerileri:
- Markdown (.md) veya PDF formatı tercih edilmeli
- Her belgeye tarih ve versiyon numarası eklenmeli
- Görsel şemalar için Canva veya Figma kullanılabilir

📄 Not:
Bu klasör, son kullanıcıya veya proje değerlendiricilerine sunulacak resmi dokümanları barındırır.
